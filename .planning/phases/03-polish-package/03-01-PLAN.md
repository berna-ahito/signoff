<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| client IP → POST /auth/login | Unauthenticated brute-force vector |
| client IP → POST /requests/ | Authenticated spam submission vector |
| client token → GET /users/ | Admin-only data; non-admin must be blocked at API layer |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-01 | Tampering | POST /auth/login brute-force | mitigate | SlowAPI 5/min per IP via @limiter.limit("5/minute") |
| T-03-02 | Denial of Service | POST /requests/ spam | mitigate | SlowAPI 20/min per IP via @limiter.limit("20/minute") |
| T-03-03 | Information Disclosure | GET /users/ user data | mitigate | require_role("admin") dependency — 403 for non-admin |
| T-03-04 | Elevation of Privilege | Non-admin accesses /users/ | mitigate | require_role("admin") in list_users() dependency |
| T-03-SC | Tampering | slowapi PyPI install | accept | Package confirmed on PyPI with 5yr history, established downloads; verified before planning |
</threat_model>
